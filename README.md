# TakeControlAPI - CSV to JSON Converter

A Python GUI application for converting CSV files to JSON format compatible with the TakeControl technician API.

## Features

- 🎯 Browse and select CSV files
- 🔐 API token management
- ✅ Email validation
- 📋 Real-time log output
- 💾 Auto-save JSON array output
- 📊 Processing statistics

## Quick Start

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python gui_converter.py`

## CSV Format

Your CSV file should have these columns:
- email
- name
- profile
- language
- timezone
